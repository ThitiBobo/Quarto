nomenclature des images:

toutes les pièces sont nommées de la façon suivante,
"piece - XXXX" où les X sont respectivement la Couleur, 
la Taille, le Troue (ou non) et la forme.

Couleurs : Rouge (R) / Bleu (B) 
Tailles  : Grand (G) / Petit (P)
Troue    : Troue (T) / Plat (P)
Forme    : Carré (C) / Rond (R)
